# 🌐 Personal Portfolio Website

## 📌 Overview

This is my personal portfolio website showcasing my projects, technical skills, education, and experience as a Computer Science student.
The website serves as a digital resume where visitors can learn more about me, explore my projects, and get in touch.

The portfolio is designed to be modern, responsive, and interactive, with features such as dark mode, animations, and smooth navigation.

---

## 🚀 Features

* Responsive design for desktop and mobile devices
* Dark mode toggle
* Animated hero section with typing effect
* Interactive project showcase
* Technical skills visualization
* Contact form for inquiries
* Smooth scrolling navigation
* Animated skill progress bars

---

## 🛠 Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript

### Libraries & Tools

* Tailwind CSS (UI styling)
* Lucide Icons (icons)
* Google Fonts

---

## 📸 Website Sections

The website includes the following sections:

* **Home** – Introduction and quick overview
* **About** – Background and personal information
* **Skills** – Technical skills and technologies
* **Projects** – Showcase of completed projects
* **Education & Experience** – Academic background and internships
* **Contact** – Ways to connect with me
